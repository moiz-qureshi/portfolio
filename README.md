# moizqureshi.com

Personal portfolio. Single HTML file, zero dependencies, no build step.

## Run locally

```
open index.html
```

## Deploy

Works on any static host. Just push and point.

| Host | How |
|------|-----|
| GitHub Pages | Settings > Pages > main branch |
| Netlify | Drag folder to [netlify.com/drop](https://app.netlify.com/drop) |
| Vercel | `npx vercel --prod` |
| Custom domain | Upload `index.html` + `og-image.png` to your server |

## What's in here

```
index.html    — the entire site
og-image.png  — social share thumbnail (1200x630)
README.md     — this file
```
