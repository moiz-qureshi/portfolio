# Portfolio

Personal portfolio site — M&A Security Lead specializing in integration and due diligence.

Single-file static site. No build step, no dependencies.

## Deploy

Drop `index.html` on any static host:

- **GitHub Pages** — Settings > Pages > Deploy from branch
- **Netlify** — Drag the folder to [app.netlify.com/drop](https://app.netlify.com/drop)
- **Vercel** — `npx vercel --prod`

## Local preview
